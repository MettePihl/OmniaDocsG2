4. Publishing¶

  1. Go into Omnia Admin.

  2. Click on “Web Content Management” in the left-hand navigation.

  3. Click on “Page Types”.

  4. Create a new page type “Web Page”.

  1. Open the tool for “Layout Templates” in the far-hand toolbar of the layout.

  2. Open the “Pages” accordion and select “Content Page - Grey”.

  1. Publish the page type.

  2. Create a new page type “News Article”.

  3. Repeat step 5 - 7, but select the template “News Page - Grey”.

  4. Click on “Business Profiles” in the left-hand navigation and select the newly created Business Profile from the previous step.

  5. Click on “Provisioning Templates” on the business profile in the left-hand navigation.

  6. Create a new template based on Publishing.

  1. Add the features “Show Business Profile Header” & “Publishing App Infrastructure” to the template.

  2. Click on “Publishing” in the left-hand navigation.

  3. Create a new Publishing App based on the newly created template.

  1. Browse to the newly created publishing app.

  2. Open the Editor and create a new Page Collection.

  1. Select the template “Intranet - Grey” under the category “Start Pages”.

  2. Go to settings and select “Web Page (Tenant)” as an available page type.

  3. Publish the page collection.

  4. Create a new flat page collection for news articles.

  1. Select the template “News Center - Grey” under the category “News Centers”.

  2. Go to settings and select “News Article (Tenant)” as an available page type.

  3. Publish the page collection.

  4. Go to settings of the publishing app and set “Start” as the default page collection.

  5. Add “Page Contact” as the Contact Property and “Review Date” as the Review Date Property.

  6. Save the settings.

  7. Go to Settings > Header for the current Business Profile.

  8. Save the Start page collection as the mega menu.
